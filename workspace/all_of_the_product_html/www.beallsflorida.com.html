Bra Measurement Guide

HOW TO MEASURE

Always measure while wearing a bra without padding.

  1. For band size hold tape gently and measure around chest directly below bust.
    • If this measurement is 33" or under ADD 5"
    • If this measurement over 33" or under ADD 3"
    • If final number is odd, round up to the next even number, (i.e., 34, 36, 38, etc.), this is the band size you should wear.
  2. For bust size, measure comfortably around fullest part of the bust.
  3. To determine cup size (i.e., A, B, C), subtract the band size measurement (including the 3-5") from the fullest bust size measurement.

For example:
If the band size is 36 (31" + 5") and the fullest bust measurement is 39", find the difference - 39"-36" = 3" or C cup. Your bra size is 36C.
